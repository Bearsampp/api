Class \doxyref{Registry}{p.}{d4/d78/class_registry}

This class provides methods to interact with the Windows \doxyref{Registry}{p.}{d4/d78/class_registry} using VBScript. It includes functionalities to check the existence of registry keys, get and set values, and delete registry entries. The class also logs operations and errors.
