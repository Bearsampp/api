\doxysubsection{Detailed Description}
Class \doxyref{Tpl\+App\+Postgresql}{p.}{d0/d74/class_tpl_app_postgresql}

This class provides methods to generate menu items and actions for managing Postgre\+SQL services within the \doxyref{Bearsampp}{p.}{da/deb/namespace_bearsampp} application. It includes functionalities for enabling/disabling Postgre\+SQL, switching versions, changing ports, changing root passwords, and managing services.
