\doxysubsection{Detailed Description}
Class \doxyref{Bin\+Nodejs}{p.}{d2/dbc/class_bin_nodejs}

The {\ttfamily \doxyref{Bin\+Nodejs}{p.}{d2/dbc/class_bin_nodejs}} class extends the {\ttfamily \doxyref{Module}{p.}{dd/df9/class_module}} class and provides functionalities specific to managing the Node.\+js module within the \doxyref{Bearsampp}{p.}{da/deb/namespace_bearsampp} application. It includes methods for reloading the module configuration, switching versions, enabling/disabling the module, and retrieving various configuration paths such as the executable, configuration file, variables file, npm executable, and launch script.
