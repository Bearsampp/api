Class \doxyref{Bin\+Php}{p.}{d2/d28/class_bin_php}

This class extends the \doxyref{Module}{p.}{dd/df9/class_module} class and provides functionalities specific to managing PHP binaries within the \doxyref{Bearsampp}{p.}{da/deb/namespace_bearsampp} application. It includes methods for reloading configurations, switching versions, updating configurations, and retrieving various PHP settings and extensions.
