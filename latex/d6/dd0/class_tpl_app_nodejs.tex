Class \doxyref{Tpl\+App\+Nodejs}{p.}{d6/dd0/class_tpl_app_nodejs}

This class provides methods to generate and manage menu items and actions related to Node.\+js within the \doxyref{Bearsampp}{p.}{da/deb/namespace_bearsampp} application. It includes functionalities for enabling/disabling Node.\+js, switching Node.\+js versions, and generating menus for Node.\+js actions.
