Class \doxyref{Tool\+Bruno}{p.}{de/dc8/class_tool_bruno}

This class represents the Bruno tool module in the \doxyref{Bearsampp}{p.}{da/deb/namespace_bearsampp} application. It extends the abstract \doxyref{Module}{p.}{dd/df9/class_module} class and provides specific functionalities for managing the Bruno IDE tool, including setting versions, and retrieving the executable path.
