\doxysubsection{Detailed Description}
Class \doxyref{Action\+Exec}{p.}{d5/d74/class_action_exec}

This class handles the execution of specific actions based on the content of a file. The actions include quitting the application or reloading it. The actions are read from a file whose path is provided by the global {\ttfamily \$bearsampp\+Core} object.
