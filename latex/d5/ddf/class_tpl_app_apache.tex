\doxysubsection{Detailed Description}
Class \doxyref{Tpl\+App\+Apache}{p.}{d5/ddf/class_tpl_app_apache}

This class provides methods to generate menu items and actions for managing Apache services within the \doxyref{Bearsampp}{p.}{da/deb/namespace_bearsampp} application. It includes functionalities for enabling/disabling Apache, switching versions, changing ports, managing modules, aliases, and virtual hosts.
