\doxysubsection{Detailed Description}
Class \doxyref{Tpl\+App\+Services}{p.}{d5/d6e/class_tpl_app_services}

This class provides methods to generate menu items and actions for managing multiple services within the \doxyref{Bearsampp}{p.}{da/deb/namespace_bearsampp} application. It includes functionalities for starting, stopping, and restarting all services at once.
