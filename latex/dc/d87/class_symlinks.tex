delete\+Current\+Symlinks (\begin{DoxyParamCaption}{}{}\end{DoxyParamCaption})\hspace{0.3cm}{\ttfamily [static]}}

Deletes all symbolic links listed in the array\+Of\+Currents. Logs each operation\textquotesingle{}s success or failure.

This method iterates over a predefined list of symbolic link paths and attempts to delete each one. If a symbolic link does not exist, an error is logged. If the deletion is successful, a debug message is logged.
