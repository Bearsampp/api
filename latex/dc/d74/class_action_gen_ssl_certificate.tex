\doxysubsection{Detailed Description}
Class \doxyref{Action\+Gen\+Ssl\+Certificate}{p.}{dc/d74/class_action_gen_ssl_certificate}

This class handles the generation of SSL certificates through a \doxyref{Win\+Binder}{p.}{d9/d34/class_win_binder} GUI. It provides a user interface for inputting the server name and target directory, and includes functionality for browsing directories, saving the certificate, and displaying progress.
