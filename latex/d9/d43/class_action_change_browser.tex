\doxysubsection{Detailed Description}
Class \doxyref{Action\+Change\+Browser}{p.}{d9/d43/class_action_change_browser}

This class handles the action of changing the default browser in the \doxyref{Bearsampp}{p.}{da/deb/namespace_bearsampp} application. It creates a window with options to select from installed browsers or browse for a different browser executable. The selected browser is then saved in the configuration.
