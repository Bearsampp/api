\doxysubsection{Detailed Description}
Class \doxyref{Tpl\+App\+Tools}{p.}{d9/ddc/class_tpl_app_tools}

This class provides methods to generate menu items and actions for managing various tools within the \doxyref{Bearsampp}{p.}{da/deb/namespace_bearsampp} application. It includes functionalities for accessing tools like Git, Python, Composer, Ghostscript, Ngrok, Pear, Perl, Ruby and more.
