\doxysubsection{Detailed Description}
Class \doxyref{Config}{p.}{dd/d34/class_config}

This class handles the configuration settings for the \doxyref{Bearsampp}{p.}{da/deb/namespace_bearsampp} application. It reads the configuration from an INI file and provides methods to access and modify these settings.
