Class \doxyref{Action\+Reload}{p.}{d3/dd8/class_action_reload}

This class handles the reloading of various configurations and settings for the \doxyref{Bearsampp}{p.}{da/deb/namespace_bearsampp} application. It performs operations such as refreshing the hostname, updating startup settings, checking and updating the browser configuration, processing configuration files, and rebuilding certain cached contents.
